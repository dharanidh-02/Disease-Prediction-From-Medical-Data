## 📌 Project Overview

CardioMind AI is a modern, production-grade **Clinical Decision Support System (CDSS)** developed as part of the **CodeAlpha Machine Learning Internship**. The application leverages a trained **Random Forest Classifier** to predict heart disease risk from patient clinical data and provides an intuitive interface for real-time medical risk assessment.

The project demonstrates the complete Machine Learning lifecycle—from data preprocessing and model training to evaluation, deployment, and an interactive web application.
